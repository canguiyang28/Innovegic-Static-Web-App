# Innovegic Ltd Professional Static Website

This package updates **innovegic_static_site_v2** with a more professional design and a transparent-background logo extracted from the uploaded image.

## Included files

- `index.html` — professional homepage for Innovegic Ltd
- `style.css` — professional responsive styling
- `innovegic-logo-transparent.png` — extracted logo with transparent background
- `innovegic-logo-header.png` — web-optimised header logo
- `social-preview.png` — optional social preview image

## Business positioning

Innovegic Ltd is a marketing and purchasing agent helping promote Chinese manufactory machinery and construction equipment to Europe, Australia, the USA and Canada.

## Recommended Azure hosting

Use **Azure Static Web Apps Free plan** for low-cost hosting, free SSL and custom domain support.

## Deploy quick steps

1. Create or open your GitHub repository.
2. Upload all files from this ZIP.
3. Azure Portal → Create → Static Web App.
4. Select the **Free** plan.
5. Connect the GitHub repository.
6. Add custom domains:
   - `innovegic.com`
   - `www.innovegic.com`
