# Innovegic Ltd bilingual website project

This project keeps the English version and adds a Chinese version in the same static website project.

Files:

- `index.html` — English homepage
- `zh.html` — Chinese homepage
- `style.css` — shared styling
- `assets/innovegic-logo-header.png`
- `assets/innovegic-logo-transparent.png`
- `assets/social-preview.png`
- `assets/whatsapp-qr.jpg`
- `assets/wechat-qr.png`

Deploy all files together. Keep the `assets` folder path unchanged.
